# Resume Upload Portal

A simple, responsive web application for uploading and displaying resumes.

## Features

- **Drag & Drop Upload**: Easily upload resume files by dragging and dropping
- **File Validation**: Supports PDF and Word documents (DOC, DOCX) up to 5MB
- **Resume Preview**: View uploaded PDF files directly in the browser
- **Download Function**: Download your uploaded resume
- **Responsive Design**: Works on desktop and mobile devices
- **Local Storage**: Resumes are stored locally in your browser

## Supported File Types

- PDF (.pdf)
- Microsoft Word (.doc, .docx)

## Usage

1. Open `index.html` in your web browser
2. Upload your resume by:
   - Clicking the upload area and selecting a file
   - Dragging and dropping a file onto the upload area
3. Click "Upload Resume" to save your file
4. View, download, or replace your resume as needed

## Files

- `index.html` - Main HTML structure
- `styles.css` - CSS styling and responsive design
- `script.js` - JavaScript functionality for file handling
- `README.md` - This documentation

## Local Development

Simply open `index.html` in any modern web browser. No server setup required.

## Browser Compatibility

Works with all modern browsers that support:
- HTML5 File API
- Local Storage
- CSS Flexbox
